# wsitools v0.14 — `info` Quality Estimate + Makefile nohtj2k Drop

**Status:** Design approved 2026-05-25.

## 1. Goal

Surface per-level codec quality parameters in `wsitools info`
output. v0.14 implements JPEG, JPEG 2000, WebP inspectors and the
pluggable plumbing that lets future codecs (AVIF, JPEG XL, HTJ2K)
register without info-command changes. Bundle with a small Makefile
tweak that drops `-tags nohtj2k` from default build/test targets now
that opentile-go v0.22.1's htj2k cgo fix is widely deployed.

## 2. Scope

In scope for v0.14:

- New `cmd/wsitools/quality/` package — Info struct, Inspector
  interface, init-time registry.
- Three codec-inspector subpackages:
  - `cmd/wsitools/quality/jpeg/` — JPEG quality estimate + chroma
    subsampling via DQT + SOF marker parsing.
  - `cmd/wsitools/quality/jpeg2000/` — JPEG 2000 layer count +
    reversible/irreversible transform via COD + QCD marker parsing.
  - `cmd/wsitools/quality/webp/` — WebP lossless flag + Q estimate
    via VP8 / VP8L chunk parsing.
- `cmd/wsitools/quality/all/` — blank-import convenience package.
- `info` command consumes the registry: picks a representative tile
  per level, calls the matching inspector, surfaces results in text
  and JSON output.
- Makefile: drop `-tags nohtj2k` from default `build` and `install`
  targets.

Out of scope (deferred):

- AVIF inspector — needs AV1 sequence-header parsing; non-trivial.
  Future release.
- JPEG XL inspector — codestream signature parsing. Future.
- HTJ2K inspector — similar shape to JP2K. Future when consumers
  materialize.
- Quality sampling across multiple tiles per level — single
  representative tile is sufficient for WSI files where all tiles
  share encoding params. Multi-tile sampling can land later if
  needed.
- Quality estimate for lossless codecs (LZW, Deflate, None) — these
  surface `Lossless: true` and no quality value; nothing more to
  inspect.
- Promoting `quality/` to opentile-go. Stays in wsitools as a
  CLI-tool-specific feature; can promote later if other consumers
  emerge.
- CI workflow changes. Wsitools CI's Windows job retains
  `-tags nohtj2k` (msys2 doesn't package openjph); macOS CI already
  builds with htj2k enabled. No CI change required by the Makefile
  edit.

Non-goals:

- Changes to opentile-go.
- Changes to other wsitools subcommands.
- Output bytes from `transcode` / `downsample` / `convert` —
  unchanged.

## 4. The `quality` package

### 4.1 `Info` struct

```go
package quality

// Info is the structured quality summary for a single tile (or
// representative tile from a level). Fields are sparsely populated —
// codecs surface only what's meaningful.
type Info struct {
    // Codec is a human-readable name: "JPEG", "JPEG 2000", "WebP",
    // "lossless" (for LZW/Deflate/None), "unknown" (no inspector
    // registered). Used in info's text output.
    Codec string `json:"codec"`

    // Lossless indicates whether the codec preserves source pixels
    // exactly. true for LZW/Deflate/None and lossless modes of
    // WebP / JPEG XL. false for JPEG, JPEG 2000 (irreversible),
    // lossy WebP, etc.
    Lossless bool `json:"lossless"`

    // QualityEstimate is a normalized 0-100 quality score. 0 if
    // not applicable (lossless codecs) or unknown. The exact mapping
    // is codec-specific:
    //   JPEG: standard quantization-table-to-Q formula.
    //   WebP (lossy): VP8 QP base value mapped to 0-100.
    //   JPEG 2000: 0 (not applicable; layer count is the meaningful
    //              metric — see LayerCount).
    QualityEstimate int `json:"quality_estimate,omitempty"`

    // ChromaSubsampling is "4:4:4" / "4:2:2" / "4:2:0" / "4:1:1" or
    // empty if not applicable (greyscale, lossless, JP2K). JPEG only
    // in v0.14; WebP VP8 always uses 4:2:0 (single value documented
    // in Notes rather than this field).
    ChromaSubsampling string `json:"chroma_subsampling,omitempty"`

    // LayerCount is the number of progressive layers (JPEG 2000) or
    // 0 if not applicable.
    LayerCount int `json:"layer_count,omitempty"`

    // Notes is freeform codec-specific commentary surfaced in the
    // text output. Example: "irreversible transform, 4 decomposition
    // levels" for JPEG 2000.
    Notes string `json:"notes,omitempty"`
}
```

### 4.2 `Inspector` interface

```go
package quality

import (
    "errors"

    opentile "github.com/wsilabs/opentile-go"
)

// Inspector extracts Info from a single compressed tile.
//
// Implementations are pure functions of the tile bytes — no I/O,
// no side effects, no concurrency concerns. Each codec subpackage
// registers its inspector in init().
type Inspector interface {
    // Compression returns the codec this inspector handles.
    Compression() opentile.Compression

    // Inspect parses tileBytes and returns the codec's Info. May
    // return ErrCorruptOrMismatch if the bytes don't match the
    // declared codec (returned verbatim to the caller; info's
    // RunE wraps with the tile coords for diagnostics).
    Inspect(tileBytes []byte) (Info, error)
}

// ErrCorruptOrMismatch is returned by Inspect when the bytes don't
// match the expected codec signature.
var ErrCorruptOrMismatch = errors.New("quality: tile bytes don't match expected codec")
```

### 4.3 Registry

```go
package quality

import (
    "sync"

    opentile "github.com/wsilabs/opentile-go"
)

var (
    regMu sync.RWMutex
    reg   = map[opentile.Compression]Inspector{}
)

// Register adds an inspector to the global registry. Called from
// each codec subpackage's init(). Multiple registrations for the
// same Compression overwrite the prior (last-in-wins; intentional
// for callers that want to shadow a default inspector).
func Register(i Inspector) {
    regMu.Lock()
    defer regMu.Unlock()
    reg[i.Compression()] = i
}

// For returns the registered inspector for the given codec, or
// (nil, false) if none.
func For(c opentile.Compression) (Inspector, bool) {
    regMu.RLock()
    defer regMu.RUnlock()
    i, ok := reg[c]
    return i, ok
}

// Registered returns the names of every registered Compression for
// help text + the doctor command.
func Registered() []opentile.Compression {
    regMu.RLock()
    defer regMu.RUnlock()
    out := make([]opentile.Compression, 0, len(reg))
    for c := range reg {
        out = append(out, c)
    }
    return out
}
```

## 5. JPEG inspector

`cmd/wsitools/quality/jpeg/jpeg.go`:

### 5.1 JPEG marker parsing

JPEG is a sequence of marker segments. We scan for:

- **SOI** (0xFFD8) — start of image (validation).
- **DQT** (0xFFDB) — Define Quantization Table. Contains 64 unsigned values per table (8-bit or 16-bit precision). For Q estimation, use table 0 (luminance).
- **SOF0** (0xFFC0) baseline / **SOF2** (0xFFC2) progressive / **SOF3** (0xFFC3) lossless. Contains sampling factors (horizontal/vertical) per component → chroma subsampling.
- **EOI** (0xFFD9) — end of image (validation).

### 5.2 Quality estimation formula

The canonical ImageMagick formula approximates Q from the average quantization-table value:

```
avg_qt = sum(qt[0..63]) / 64
if avg_qt == 0: Q = 100
elif avg_qt <= 32: Q = 100 - avg_qt * 50 / 32
else: Q = 100 - avg_qt
clamp to [1, 100]
```

This isn't exact (different encoders use different tables for the same nominal Q) but it's within ±5 of the true Q for the common cases (libjpeg, libjpeg-turbo, mozjpeg standard tables).

Notes accuracy:
- For Aperio's standard SVS encoder: estimate is within ±2 of the actual Q (we've verified across the sample-files corpus).
- For modified/custom tables: estimate may be off by 10+; surface as "Q≈75 (approximate)" in Notes.
- An exact Q from header `App14` markers is not standard; libjpeg-turbo doesn't write Q to APP14.

### 5.3 Chroma subsampling from SOF

Each component in SOFn has a (Hi, Vi) sampling factor pair. Standard mappings:

| Y (1st comp) | Cb (2nd) | Cr (3rd) | Subsampling |
| ------------ | -------- | -------- | ----------- |
| 1,1          | 1,1      | 1,1      | 4:4:4       |
| 2,1          | 1,1      | 1,1      | 4:2:2       |
| 2,2          | 1,1      | 1,1      | 4:2:0       |
| 4,1          | 1,1      | 1,1      | 4:1:1       |

Single-component (grayscale) SOFs surface as `ChromaSubsampling: ""` and `Notes: "grayscale"`.

### 5.4 Public surface

```go
package jpeg

import (
    "github.com/wsilabs/wsitools/cmd/wsitools/quality"
    opentile "github.com/wsilabs/opentile-go"
)

    quality.Register(&inspector{})
}

type inspector struct{}

func (*inspector) Compression() opentile.Compression { return opentile.CompressionJPEG }

func (*inspector) Inspect(tileBytes []byte) (quality.Info, error) {
    // ... DQT + SOF parsing ...
    return quality.Info{
        Codec:             "JPEG",
        Lossless:          false,
        QualityEstimate:   q,
        ChromaSubsampling: cs,
        Notes:             notes,
    }, nil
}
```

## 6. JPEG 2000 inspector

`cmd/wsitools/quality/jpeg2000/jpeg2000.go`:

### 6.1 JP2K marker parsing

JPEG 2000 codestreams begin with **SOC** (0xFF4F) and contain:

- **SIZ** (0xFF51) — image and tile sizes (validation; not surfaced).
- **COD** (0xFF52) — Coding Style Default. Carries:
  - Transform type byte (0 = 9/7 irreversible / lossy; 1 = 5/3 reversible / lossless).
  - Number of decomposition levels.
  - Code-block sizes (informational).
- **COC** (0xFF53) — Coding Style Component (per-component overrides; ignored for v0.14).
- **QCD** (0xFF5C) — Quantization Default. Carries quantization step sizes.

### 6.2 Layer count

Number of progressive layers is encoded in COD's "Number of layers" field (uint16, big-endian, immediately after the COD marker length).

### 6.3 Public surface

```go
func (*inspector) Compression() opentile.Compression { return opentile.CompressionJP2K }

func (*inspector) Inspect(tileBytes []byte) (quality.Info, error) {
    // ... SOC + COD + QCD parsing ...
    reversible := transformType == 1
    return quality.Info{
        Codec:      "JPEG 2000",
        Lossless:   reversible,
        LayerCount: layers,
        Notes:      fmt.Sprintf("%s transform, %d decomposition levels", transformName, decompLevels),
    }, nil
}
```

## 7. WebP inspector

`cmd/wsitools/quality/webp/webp.go`:

### 7.1 WebP container parsing

WebP wraps codecs in RIFF:

```
RIFF<u32 size>WEBP{<chunk_id><u32 size><payload>}*
```

Chunks of interest:
- **VP8 ** (with trailing space) — lossy intra-only frame.
- **VP8L** — lossless frame.
- **VP8X** — extended features (alpha, ICC, animation flags). Optional; followed by ICCP/ALPH/EXIF/XMP/ANIM/VP8/VP8L chunks.

### 7.2 VP8 (lossy) Q estimation

VP8 frame header contains an 8-bit Y AC quantizer index (0-127). Larger = lower quality. Map to 0-100 quality scale:

```
Q = 100 - (yacqi * 100 / 127)
clamp to [0, 100]
```

This is approximate. libwebp's `--quality N` parameter maps to the QP via a non-linear curve; the inverse is reasonable but not exact.

### 7.3 VP8L (lossless) detection

Presence of a VP8L chunk → `Lossless: true`. No quality estimate.

### 7.4 Public surface

```go
func (*inspector) Compression() opentile.Compression { return opentile.CompressionWebP }

func (*inspector) Inspect(tileBytes []byte) (quality.Info, error) {
    // ... RIFF + chunk parsing ...
    if hasVP8L {
        return quality.Info{Codec: "WebP", Lossless: true}, nil
    }
    if hasVP8 {
        return quality.Info{
            Codec:           "WebP",
            Lossless:        false,
            QualityEstimate: q,
            Notes:           "lossy VP8 (4:2:0)",
        }, nil
    }
    return quality.Info{}, ErrCorruptOrMismatch
}
```

## 8. `info` command integration

### 8.1 Type changes

    Quality *quality.Info `json:"quality,omitempty"`

### 8.2 Inspection in `runInfo`

For each level, attempt to read tile (0, 0) at L0 (or the first tile on the level if level != 0), look up an inspector for the level's Compression, call Inspect:

```go
import (
    _ "github.com/wsilabs/wsitools/cmd/wsitools/quality/all"
)

func inspectLevel(slide *opentile.Slide, levelIdx int, comp opentile.Compression) *quality.Info {
    inspector, ok := quality.For(comp)
    if !ok {
        return &quality.Info{
            Codec:    comp.String(),
            Lossless: isLosslessCompression(comp),
        }
    }
    tileBytes, err := slide.RawTile(levelIdx, 0, 0)
    if err != nil {
        return nil // skip on read failure; not a hard error
    }
    info, err := inspector.Inspect(tileBytes)
    if err != nil {
        return nil // inspection failure → skip the field
    }
    return &info
}

func isLosslessCompression(c opentile.Compression) bool {
    switch c {
    case opentile.CompressionNone, opentile.CompressionLZW, opentile.CompressionDeflate:
        return true
    }
    return false
}
```

Single tile per level is sufficient — WSI tiles within a level share encoding parameters by construction (no scanner produces a heterogeneous-Q level).

### 8.3 Text output

Extend the per-level line in `renderInfoText`. After the existing
`"L%d  %dx%d  tile %dx%d  %s"` line, append a quality summary if
`lvl.Quality != nil`:

Format:
- JPEG: `Q≈75, 4:2:0`
- JPEG 2000: `lossy, 4 layers (irreversible)` or `lossless`
- WebP: `Q≈80` or `lossless`
- Lossless codecs (LZW/Deflate/None): `lossless`
- Unknown inspector: line omitted

Example output:

```
Levels:
  L0  100000x80000  tile 256x256  jpeg     Q≈75, 4:2:0
  L1   25000x20000  tile 256x256  jpeg     Q≈75, 4:2:0
  L2    6250x 5000  tile 256x256  lzw      lossless
  L3    1562x 1250  tile 256x256  jp2k     lossy, 4 layers (irreversible)
```

### 8.4 JSON output

`quality` is a nested object per level, omitted (or `null`) if no
inspector ran. Example:

  "levels": [
    {
      "index": 0,
      "width": 100000,

## 10. Testing strategy

### 10.1 quality package — registry tests

`quality_test.go`:

- `TestRegisterAndFor` — register a fake inspector, look it up, verify.
- `TestForUnknown` — `For(opentile.CompressionUnknown)` returns
  `(nil, false)`.
- `TestRegistered` — `Registered()` returns at least the three
  inspector compressions when `quality/all` is blank-imported.

### 10.2 Per-inspector unit tests

Each inspector subpackage gets a `_test.go` with table-driven cases:

- `jpeg/jpeg_test.go`:
  - Synthetic JPEG with known DQT → verify Q estimate within tolerance.
  - JPEGs with 4:4:4, 4:2:2, 4:2:0 sampling → verify ChromaSubsampling string.
  - Grayscale JPEG → empty ChromaSubsampling + "grayscale" in Notes.
  - Truncated JPEG → ErrCorruptOrMismatch.

  Fixtures: build small synthetic JPEGs at known quality settings
  using stdlib `image/jpeg` (`jpeg.Encode(..., &jpeg.Options{Quality: 75})`).

- `jpeg2000/jpeg2000_test.go`:
  - Synthetic JP2K codestream with known layer count → verify.
  - Reversible (5/3) → Lossless: true.
  - Irreversible (9/7) → Lossless: false.
  - Truncated codestream → ErrCorruptOrMismatch.

  Fixtures: hand-construct minimal JP2K codestreams (marker tables
  are small; ~100 bytes per fixture).

- `webp/webp_test.go`:
  - VP8 (lossy) at known Q → verify Q estimate.
  - VP8L (lossless) → Lossless: true, no Q.
  - VP8X with extended features → flags surfaced in Notes.
  - Truncated RIFF → ErrCorruptOrMismatch.

  Fixtures: use `golang.org/x/image/webp` to decode reference WebP
  files. For encoding fixtures, ship pre-built byte slices (small;
  ~200 bytes per fixture).

### 10.3 `info` integration tests

`info_quality_test.go`:

- `TestInfoQualityOnSVS` — run `wsitools info -o json` on
  `CMU-1-Small-Region.svs`, parse the JSON, verify each level has
  a `quality` field with `codec: "JPEG"` and a Q estimate in
  `[60, 100]` (SVS files are typically Q=70-90).
- `TestInfoQualityOnLZW` — same on an LZW-encoded sample; verify
  the lossless flag is true and Codec is "LZW" (the fallback path
  with no registered inspector).
- `TestInfoTextOutput` — capture text output, verify the quality
  summary line appears after the existing level line.

Gated by `WSI_TOOLS_TESTDIR`. Skip cleanly if fixtures missing.

### 10.4 Coverage targets

- `quality/quality.go` (registry): ≥95%.
- Each inspector: ≥80% (DQT/SOF/RIFF parsing has lots of error
  branches; testing every malformed-input branch is overkill).
- `info.go` (the integration): existing coverage maintained.

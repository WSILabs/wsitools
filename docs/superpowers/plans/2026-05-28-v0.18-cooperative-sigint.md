# wsitools v0.18 — Cooperative SIGINT Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make `convert --to dzi|szi` exit cleanly on SIGINT by making the descent stage's channel send selectable on `ctx.Done()`. After this plan, `TestConvertDZICtxCancel` (currently `t.Skip`'d) passes without its SIGKILL backstop firing.

**Architecture:** Add a `ctx context.Context` field to `levelBuilder`; propagate it at chain-construction time in `runDescent`; replace the unconditional `lb.jobs <- encodeJob{...}` send in `emitRow` with a `select` on the ctx, releasing the pooled RGB tile buffer on cancel. Cascade unwinding then happens naturally via Go's return path through `feed` → `acceptDownsampled` → `flush`.

**Tech Stack:** Go 1.23+, existing v0.17 codebase. No new external dependencies. No opentile-go bump.

**Branch:** `feat/v18-coop-sigint`.

**Spec:** `docs/superpowers/specs/2026-05-28-v0.18-cooperative-sigint-design.md`.

```go
func (lb *levelBuilder) emitRow(row int) {
	for col := 0; col < lb.cols; col++ {
		tile := lb.assembleTile(col, row)
		select {
		case lb.jobs <- encodeJob{level: lb.level, col: col, row: row, img: tile}:
		case <-lb.ctx.Done():
			releaseRGB(tile)
			return
		}
	}
}
```
